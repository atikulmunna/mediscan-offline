# Dataset Notes

Use this file to track anything useful about the reference images.

Suggested fields per medicine set:

- medicine name
- generic name
- manufacturer
- files available
- expected batch number
- expected expiry date
- notes about blur, glare, crop, or low light

Example:

```text
Medicine: Napa
Generic: Paracetamol
Manufacturer: Beximco
Files: front.jpg, back.jpg, side-expiry.jpg
Expected batch: BX-1234
Expected expiry: 2027-01
Notes: expiry text is faint on side panel
```
